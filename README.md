# Config Files

Personal dotfiles I sync across machines.

- [`~/.gitconfig`](.gitconfig) - Git
- [`~/.vimrc`](.vimrc) - Vim
- [`~/.config/nvim/init.lua`](.config/nvim/init.lua) - Neovim
- [`keybindings.json`](keybindings.json) - VS Code keymap inspired by LazyVim motions with my own tweaks. Highly recomended to combine it with vim motions extension.
